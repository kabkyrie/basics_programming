***Информация*** - данные, знания, сведения в том или ином виде (также ***информация*** - степень уменьшения неопределенности)

Обычно для измерения количества информации берут $log_2(b)$ или $log_{256}(b)$, где $b$ - биты и байты соответственно

---
### Представление синтаксиса и семантики

***Данные*** - представление информации
***Синтаксис*** - визуальное представление информации
***Семантика*** - смысл и трактование данного представления информации

---
### Количество информации
***Количество информации*** - это числовая характеристика, отражающая степень неопределенности, которая исчезнет после получения данной информации

|          | обозначение | степень  |
| -------- | ----------- | -------- |
| Байт     | -           | $2^{0}$  |
| Килобайт | KiB         | $2^{10}$ |
| Мегабайт | MiB         | $2^{20}$ |
| Гигабайт | GiB         | $2^{30}$ |
| Терабайт | TiB         | $2^{40}$ |

---
### Свойства информации

1) **Объективность** - независимость от человеческого фактора
2) **Достоверность** - если отражает истинное положение дел
3) **Полнота** - если ее достаточно для понимания и принятия решений
4) **Точность** - степень ее близости к реальному состоянию объекта, процесса, явления
5) **Актуальность** - важность для настоящего времени
6) **Понятность** - если она выражена языком, на котором говорят те, кому предназначена эта информация
7) **Доступность** - возможность получения информации
