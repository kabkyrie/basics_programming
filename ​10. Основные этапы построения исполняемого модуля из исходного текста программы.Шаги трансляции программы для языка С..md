
### Основные этапы построения исполняемого модуля из исходного текста программы на примере gcc c

#### 1) Препроцессор (утилита cpp)
Подготавливает код программы на языке С к компиляции 
- Обрабатывает директивы ```#``` (```#include ```, ```#defin``` и тд)
- Выполняет макро-подстановку (замена сокращений на код)
- Удаляет комментарии
Результат: Полный текст программы (единица трансляции) без директив.

#### 2) Компилятор (утилита cc1)
Перевод кода с высокоуровневого языка на язык Ассемблера и состроит из фаз компилятора

1) **Лексический анализ** (сканер / tokinizer)
	- Разбивает поток символов на смысловые единицы - Токены (ключевые слова. операторы, идентификаторы, литералы)
2) **Синтаксический анализ**
	- Проверяет грамматическую структуру
	- Строит дерево разбора из токенов (Syntax tree)
3) **Семантический анализ**
	- Проверяет смысл операций (совместимость типов, видимость переменных)
	- В итоге получается проверенное дерево
4) **[[9. Транслятор. Интерпретатор. Компилятор. JIT-компилятор. Оптимизации кода. Проекции Футамуры-Ершова#^a6383b | Оптимизация]]**
	- Улучшает код (удаляет лишнее, упрощает вычисления)
	- В итоге получается аккуратное дерево
5) **Генерация**
	- Транслирует дерево в низкоуровневые инструкции
	- Результат: текст на языке Ассемблера

#### 3) Ассемблер (утилита gas)
Специфический транслятор языка мнемоник
- Переводит инструкции ассемблера в машинный код
- Результат: **Объектный модуль** (файл .o). Он содержит только машинный код без адресов внешних функций.

#### 4) Линкер (редактор связей / утилита ld)
Финальный этап сборки
- Объединяет несколько объектных модулей в один
- Подключает статические и динамические библиотеки
- Разрешает ссылки (связывает вызов функции в одном файле с её кодом в другом)

Работа линкера заключается в определении в каждом объектном модуле ссылок на неопределенные имена, заменяя его на адрес в других объектных модулях.

**Результат:** Исполняемый файл (Executable) или динамическая библиотека

---

### Виды ошибок

1) Ошибка препроцессора
	- Не найдет файл ```#include ...``` ошибка в макросе
2) Ошибки компилятора
	- **Лексические:** недопустимый символ
	- **Синтаксические:** пропущена точка с запятой, скобка
	- **Семантические:** попытка сложить число и строку, использование необъявленной переменной 
3) Ошибки линкера
	- ```Unresolved external symbol``` (функция объявлена, но ее код не найден ни в одном из модулей или библиотек)
